协作翻译须知

感谢各位参与协作翻译，我们可以一边学习，一边翻译，取得更好的学习效果。

你可以翻译整个章节，也可以翻译一个段落。在自己要翻译的部分标注，表明领取任务。在自己翻译的部分上面和下面加上分隔线。也可以对已有的译文利用“评论”功能进行讨论，自己可以在不确定的地方“评论”。可以参照这个正在翻译的文章：https://shimo.im/doc/CZU87ZsZO38SF0vM 

翻译初稿结束后，会有校对。
